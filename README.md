# Sistema Integral de Gestión de Clientes, Servicios y Reservas - Software FJ

## Descripción del proyecto

Este proyecto corresponde al desarrollo de una aplicación orientada a objetos en Python para la empresa ficticia Software FJ. El sistema permite gestionar clientes, servicios y reservas sin utilizar bases de datos, implementando principios fundamentales de programación orientada a objetos y manejo avanzado de excepciones.

La aplicación fue desarrollada como parte del curso de Programación de la Universidad Nacional Abierta y a Distancia (UNAD).

---

## Características principales

* Registro de clientes
* Gestión de servicios
* Creación de reservas
* Manejo de excepciones personalizadas
* Registro de errores en archivo logs.txt
* Arquitectura modular
* Uso de listas internas
* Aplicación estable y robusta
* Programación orientada a objetos

## Funcionamiento del sistema

El sistema funciona mediante un menú interactivo en consola que permite:

1. Registrar clientes
2. Visualizar servicios disponibles
3. Crear reservas
4. Consultar reservas realizadas
5. Gestionar errores sin detener la aplicación
